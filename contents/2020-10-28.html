      # 关于函数重载

      # 问题

      JavaScript不支持函数重载,但TypeScript是否支持函数重载?

      # 答案

      支持,但是它的实现很奇怪,当我们在TypeScript中执行函数重载时,我们只能实现一个带有多个签名的函数

      // 带有数字类型参数的函数
function add(a:string,b:string):string;
// 带有数字类型参数的函数
function add(a:number,b:number):number;

// 函数定义

function add(a:any,b:any):any{
  return a + b
}
