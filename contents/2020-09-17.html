      # 答案

      a -> Promise -> undefined -> hello -> la -> undefind -> zzz

      # 解析

      • async函数执行的返回结果是一个promise对象,这个函数的返回值是这个promise状态值resolve的值

      • await后面如果不是一个promise对象,将直接返回这个值

      • await后面如果是一个promise对象.将会把这个promise的状态resolve返回出去

      以上没有考虑reject状态
